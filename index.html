Termos de Privacidade

1. Introdução

Boas-vindas à extensão CardSage. Estamos comprometidos em proteger a sua privacidade e garantir que seus dados sejam tratados com segurança. Esta página descreve como coletamos, utilizamos e protegemos suas informações quando você usa nossa extensão.

2. Coleta de Dados

2.1 Dados Coletados

A extensão coleta os seguintes dados:

  • Dados de Texto da Página: Quando solicitado pelo usuário, a extensão pode acessar e manipular os dados de texto da página que o usuário está visitando.
  • Lista de Favoritos: A extensão permite que o usuário salve uma lista de favoritos escolhida por ele.

2.2 Dados Não Coletados

A extensão não coleta, armazena ou processa os seguintes tipos de dados:

  • Dados sensíveis (informações pessoais, financeiras, de saúde, etc.).
  • Dados de navegação do usuário fora das interações explícitas com a extensão.

3. Uso dos Dados

Os dados coletados pela extensão são utilizados para os seguintes propósitos:

  • Manipulação de Dados de Texto: A extensão manipula os dados de texto da página conforme solicitado pelo usuário.
  • Armazenamento de Favoritos: A extensão armazena a lista de favoritos escolhida pelo usuário para facilitar o acesso futuro.

4. Compartilhamento de Dados

A extensão não compartilha, vende ou divulga quaisquer dados coletados a terceiros. Todos os dados permanecem no dispositivo do usuário e são utilizados exclusivamente para os fins descritos nesta política.

5. Segurança dos Dados

Adotamos medidas de segurança apropriadas para proteger os dados contra acesso não autorizado, alteração, divulgação ou destruição. Embora nos esforcemos para usar meios comercialmente aceitáveis para proteger seus dados, não podemos garantir sua segurança absoluta.

6. Direitos dos Usuários

Os usuários têm o direito de:

  • Acessar: Solicitar informações sobre os dados que a extensão coletou.
  • Corrigir: Solicitar a correção de quaisquer dados incorretos.
  • Excluir: Solicitar a exclusão de seus dados armazenados pela extensão.

7. Alterações a Esta Política

Podemos atualizar nossa política de privacidade periodicamente. Notificaremos os usuários sobre quaisquer mudanças publicando a nova política nesta página. Recomendamos que você revise esta política regularmente para quaisquer alterações.
